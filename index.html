QuickSquare 1.0

by Will Leinweber

My use-case for square is really just going out to eat with friends and saving the day when no one has cash. QuickSquare calculates the processing fees so I can pass them on ;)

It's also mostly an excuse to buld an html5 iphone app. The source is on github at github.com/will/quicksquare, and the latest version of this app should be at will.github.com/quicksquare.
